A complete production-grade learning repository for mastering modern software engineering, AI infrastructure, and intelligent agentic systems. Built with deep **Hinglish explanations**, production-ready examples, architecture diagrams, debugging guides, scalability patterns, and real-world engineering concepts.

---

## 🤖 What is SusaGPT?

**SusaGPT** was initially conceived as an ambitious project to build a high-performance LLM from scratch. We successfully implemented the core architecture, training loops, and tokenizer logic (found in the `docs/exercises/` and `docs/susagpt/` directories).

However, due to the limited availability of high-quality, diverse training data required for a state-of-the-art model, the model development reached a plateau. Rather than leaving the repository incomplete, we decided to transform it into the **Ultimate Learning Operating System**. 

We have now populated this repository with the **Full Engineering Curriculum** — covering everything from AI Foundations and LLM Research to System Design and Full Stack Mastery — to ensure that every developer can learn the exact skills needed to build systems as complex as SusaGPT themselves.
