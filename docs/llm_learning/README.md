This folder contains a structured, concept-first learning path for large language models (LLMs). It covers foundations, architecture, data and training, deployment, evaluation, safety, tooling, and real-world applications. The material is designed to minimize heavy code blocks and maximize clear conceptual understanding, with practical examples where helpful.

Curriculum outline (levels hint):
- Foundations: core concepts, tokens, transformers, training objectives.
- Architecture & Training: model architectures, scaling laws, data pipelines, fine-tuning strategies.
- Inference & Deployment: decoding strategies, serving, latency, monitoring.
- Evaluation & Benchmarks: metrics, evaluation suites, human-in-the-loop.
- Safety & Alignment: policy, red-teaming, guardrails.
- Tools & APIs: popular frameworks, libraries, datasets, and orchestration tooling.
- Applications & Capstone Projects: end-to-end prompts, retrieval augmented generation, and building demos.

Usage tips:
- Read topics in order, then skim ahead to advanced topics as needed.
- When code is shown, focus on the underlying concept rather than boilerplate.
- Propose hands-on exercises at the end of each module to reinforce learning.
