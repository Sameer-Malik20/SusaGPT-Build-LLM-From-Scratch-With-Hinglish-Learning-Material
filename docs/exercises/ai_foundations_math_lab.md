## 🚀 Exercise 2: Calculus — The Engine of Learning
**Problem Statement:**
Aapko ek simple cost function diya gaya hai: `f(x) = x^2 + 5x + 6`.
1. Iska derivative `f'(x)` calculate karo.
2. **Gradient Descent**: Ek loop likho jo function ke "Minimum" value tak pahunche (Learning rate use karke).
3. Visualize karo ki gradient kaise shift hota hai har iteration mein.

## 🏗️ Exercise 3: Probability & Information Theory
**Problem Statement:**
Aapka model predict kar raha hai ki image "Cat" hai (80%) ya "Dog" hai (20%).
1. Calculate the **Entropy** of this distribution.
2. **Cross-Entropy Loss**: Calculate the loss if the actual label was "Dog."
3. Discussion: "Why is Cross-Entropy better than Mean Squared Error for classification?"
